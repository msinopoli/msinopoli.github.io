CSS Selectors

Universal * Selects all elements of any type
Type p, h1, h2 Selects all elements of speceifc type
Class .title, .example Selects all elements with that specific class
ID #title, #example Selects all elements with that specific ID
Attribute img[src*='example'] Selects all the elements with that specific attribute
Type Examples Description

*Note the order of this list matters. Each of selectors will overide the next one. For example if you have all h3 headings set to the color blue but you have a specific h3 heading with the id special set to the color orange, then that h3 heading will become orange while the rest remain blue
